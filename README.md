# Intelligent-PDF-Query-System
A smart AI tool acts as an interactive chatbot for any lengthy PDF. You can simply upload a document and ask questions in natural language. Using a RAG pipeline and semantic search, it instantly finds exact, hallucination-free answers from the text. It also remembers your previous queries. The similar search results are passed as context to the LLM toh generate the response making document analysis totally effortless.


using Python for Back-end development
Streamlit for Front-end
Faiss as vector database
ollama for embeddings
groq as llm
